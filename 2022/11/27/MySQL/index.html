发表于 2022-11-27

Mysql安装教程

第一步

下载压缩包到你的安装路径（就是你想把mysql安装到哪里，你就放在那里），然后解压

这里我想把mysql安装到D盘，所以我把压缩包安装到D盘，并且解压到D盘，此时压缩包已经没有用了，可以把他删掉了。

第二步

我们进入解压后的文件

右键点击，新建文本文档文件，然后将名字改成imi，将后缀名改成ini

新建文本文档就是他的名字，后面的txt就是他的后缀

改完后，我们打开这个my.ini文件

将以下内容复制到这个文件中,然后保存

需要注意的是:

dasedir=为你的解压后的文件的路径，例如我的解压文件路径是：D:\mysql-8.0.11-winx64，所以我的basedir=后面为：D:\\mysql-8.0.11-winx64 ，要注意的是，有\的地方再加个\,例如我的D:后面有个\，所以我输入成D:\\

datadir=你的解压后的文件的路径\\data，例如我的解压文件路径是：D:\mysql-8.0.11-winx64，所以我的datadir=后面为：D:\\mysql-8.0.11-winx64\\data ，要注意的是，有\的地方再加个\,例如我的D:后面有个\，所以我输入成D:\\，我的mysql-8.0.11-winx64后面有个\，所以我输入成：mysql-8.0.11-winx64\\

第三步 配置环境变量

打开设置

打开系统：

点击关于

点击高级系统设置：

点击环境变量：

点击系统变量下面的新建

变量名输入：Path

变量值输入;安装路径下bin文件夹的路径; （注意前后加 ; ）

然后点击确定

第四步 初始化数据库文件目录

点击win键，就是你的键盘上的最底下的那个有四个方块的键

输入cmd

点击以管理员身份运行，注意！！！！ 点击以管理员身份运行，点击以管理员身份运行

打开之后，如果安装到D盘，输入: d: ,然后回车（按Enter键）

如果安装到c盘，请忽略这一步

然后输入cd 你的安装包中bin文件夹的路径

我的安装包的路径为：D:\mysql-8.0.11-winx64，所以我输入 cd D:\mysql-8.0.11-winx64\bin

初始化命令：mysqld –initialize

然后继续输入：mysqld –initialize

等执行结束之后，不要叉掉你的命令提示符，你再次打开并进入你的解压后的文件，文件结构如下：

此时我们会发现多了一个data文件，点进去后，我们发现生成了一下文件，证明执行成功

第五步 下载并启动mysql服务

我们返回命令提示符，输入：mysqld -install

执行完毕后，会打印出：Service successfully installed

然后关闭命令提示符，重新以管理员身份打开命令提示符

输入：net start mysql

第六步 进入mysql，并修改密码

然后输入：mysql -u root -p

然后我们会发现提示你输入密码（Enter password:），我们打开你的解压后的文件，进入data文件夹

找到文件后缀为err的文件，以记事本打开

打开后我们找到有 root@localhost的这一行

root@localhost：后面的为你的初始化密码，例如我的初始化密码为：I(oz,-Xye0hW

然后我们将密码仔细的输入到命令行中，记住要手动输入，然后按回车

当看到mysql>,则进入了mysql当中

最后我们修改密码，输入：ALTER USER ‘root‘@’localhost’ IDENTIFIED BY ‘密码’;

然后回车

例如我的密码想修改为：123456，

所以我输入：ALTER USER ‘root‘@’localhost’ IDENTIFIED BY ‘123456’;

注意：最后要加 ;

现在，我们已经安装完mysql了

我们退出mysql，输入：exit，然后回车

当我们想进入mysql的时候

输入：mysql -u root -p

然后回车，输入你修改完的密码

就可以进入了

如果不想让数据库系统运行，可以停止mysql服务

停止mysql服务也是在管理员下的命令行界面下执行：net stop mysql

也可以用图形界面方式启动和停止mysql服务，方法如下：
右击”计算机“ —> ‘管理’ —> ‘服务与应用程序’ —> ‘服务’，在服务窗口中找到‘mysql’服务，右击它，启动或停止
